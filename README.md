### 2. Создание репозитория и дбавление файлов

```
git init

git add text.txt
git add .
git status
```

### 4. HEAD - всему голова


Файл HEAD (англ. «голова», «головной») — один из служебных файлов папки .git. 
Он указывает на коммит, который сделан последним (то есть на самый новый).
