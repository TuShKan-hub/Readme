## Введение


1. Терминал Bush(Основные команды)
2. Создание репозитория и дбавление файлов

---

### 1 Пункт


```
mkdir my_project
cd my_project
touch text.txt
cd ~; /; ..;
```


---

### 2 Пункт


```
git init
git add --all
git add text.txt
git add .
git status
```


---


### 3 Пункт


```
git commit -m 'Мой первый коммит'
```
